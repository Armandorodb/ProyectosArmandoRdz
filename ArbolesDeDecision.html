Rendimiento del combustible en autos comerciales.

Realizaremos un anlisis de diferentes vehculos para poder realizar la prediccion sobre si la cantidad de cilindros de un auto es mayor a 4 cilindros(1) o menor o igual a 4 cilindros(0) a travs de, la funcin logistica, arboles de decisin y de la funcin discriminante.

Para realizar dicho estudio obtuvimos informacin del UCI Machine Learning Repository especficamente de Auto MPG de un estudio de Quinlan, R. (1993).

Dicha informacin cuenta con las siguientes variables:

MPG: Miles per galon, millas por galn, nuestra variable de inters.
Cylinders: La cantidad de cilindros del auto.
Displacement: Volumen total de aire y combustible que un auto puede desplazar en un ciclo de combustin.
Horsepower: Caballos de fuerza, es una medida de potencia.
Weight: Peso en libras.
Acceleration: Aceleracin.
Model year: Ao del modelo de auto.
Origin: Pas de origen del auto. 1: America, 2: Europa, 3: Asia
Car Name: Modelo del auto.

La base de datos tena algunos valores de la variable de inters como NaN, se eliminaron esas filas ya que considero que cualquier manera de conseguir ese valor aadira incertidumbre a nuestro modelo. Estaramos construyendo nuestro modelo en terreno que no es firme.

Ajustaremos los datos para poder realizar nuestro objetivo.

from sklearn.metrics import accuracy_score

# Usamos ccp_alpha para la poda
alphas = np.arange(0.01, 0.1, 0.01)  # Probamos con menos valores de  para hacerlo ms rpido

# Restablecemos el proceso de bsqueda de 
best_alpha = None
best_accuracy = 0

# Bsqueda del mejor  mediante validacin cruzada con los datos de entrenamiento
for alpha in alphas:
    accuracies = []
    
    # Entrenamos el modelo con los datos de entrenamiento (X_train2, y_train2)
    clf = DTC(ccp_alpha=alpha)
    clf.fit(x_train2, y_train2)
    y_pred = clf.predict(x_test2)  # Realizamos predicciones con los datos de prueba
    accuracies.append(accuracy_score(y_test2, y_pred))  # Calculamos la precisin
    
    # Calculamos la media de la precisin para el valor de  actual
    mean_accuracy = np.mean(accuracies)
    
    if mean_accuracy > best_accuracy:
        best_accuracy = mean_accuracy
        best_alpha = alpha

# Entrenamos el modelo con el mejor  encontrado
clf = DTC(ccp_alpha=best_alpha)
clf.fit(x_train2, y_train2)  # Entrenamos con los datos de entrenamiento

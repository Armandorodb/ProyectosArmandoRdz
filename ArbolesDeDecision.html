Rendimiento del combustible en autos comerciales.¶

Realizaremos un análisis de diferentes vehículos para poder realizar la prediccion sobre si la cantidad de cilindros de un auto es mayor a 4 cilindros(1) o menor o igual a 4 cilindros(0) a través de, la función logistica, arboles de decisión y de la función discriminante.

Para realizar dicho estudio obtuvimos información del UCI Machine Learning Repository específicamente de Auto MPG de un estudio de Quinlan, R. (1993).

Dicha información cuenta con las siguientes variables:

MPG: Miles per galon, millas por galón, nuestra variable de interés.
Cylinders: La cantidad de cilindros del auto.
Displacement: Volumen total de aire y combustible que un auto puede desplazar en un ciclo de combustión.
Horsepower: Caballos de fuerza, es una medida de potencia.
Weight: Peso en libras.
Acceleration: Aceleración.
Model year: Año del modelo de auto.
Origin: País de origen del auto. 1: America, 2: Europa, 3: Asia
Car Name: Modelo del auto.

La base de datos tenía algunos valores de la variable de interés como NaN, se eliminaron esas filas ya que considero que cualquier manera de conseguir ese valor añadiría incertidumbre a nuestro modelo. Estaríamos construyendo nuestro modelo en terreno que no es firme.

Ajustaremos los datos para poder realizar nuestro objetivo.

from sklearn.metrics import accuracy_score

# Usamos ccp_alpha para la poda
alphas = np.arange(0.01, 0.1, 0.01)  # Probamos con menos valores de α para hacerlo más rápido

# Restablecemos el proceso de búsqueda de α
best_alpha = None
best_accuracy = 0

# Búsqueda del mejor α mediante validación cruzada con los datos de entrenamiento
for alpha in alphas:
    accuracies = []
    
    # Entrenamos el modelo con los datos de entrenamiento (X_train2, y_train2)
    clf = DTC(ccp_alpha=alpha)
    clf.fit(x_train2, y_train2)
    y_pred = clf.predict(x_test2)  # Realizamos predicciones con los datos de prueba
    accuracies.append(accuracy_score(y_test2, y_pred))  # Calculamos la precisión
    
    # Calculamos la media de la precisión para el valor de α actual
    mean_accuracy = np.mean(accuracies)
    
    if mean_accuracy > best_accuracy:
        best_accuracy = mean_accuracy
        best_alpha = alpha

# Entrenamos el modelo con el mejor α encontrado
clf = DTC(ccp_alpha=best_alpha)
clf.fit(x_train2, y_train2)  # Entrenamos con los datos de entrenamiento

# Visualizamos el árbol de decisión resultante

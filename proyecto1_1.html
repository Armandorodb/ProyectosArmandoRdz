Rendimiento del combustible en autos comerciales.

Realizaremos un anlisis de diferentes vehculos para poder realizar inferencias sobre su rendimiento de combustible, especficamente la cantidad de millas que pueden recorrer por galn de gasolina, adems de las inferencias buscamos realizar un modelo que sea capaz de predecir con cierta certeza la cantidad de millas que recorre un auto con conocer cierta informacin de este.

Para realizar dicho estudio obtuvimos informacin del UCI Machine Learning Repository especficamente de Auto MPG de un estudio de Quinlan, R. (1993).

Dicha informacin cuenta con las siguientes variables:

MPG: Miles per galon, millas por galn, nuestra variable de inters.
Cylinders: La cantidad de cilindros del auto.
Displacement: Volumen total de aire y combustible que un auto puede desplazar en un ciclo de combustin.
Horsepower: Caballos de fuerza, es una medida de potencia.
Weight: Peso en libras.
Acceleration: Aceleracin.
Model year: Ao del modelo de auto.
Origin: Pas de origen del auto. 1: America, 2: Europa, 3: Asia
Car Name: Modelo del auto.

Creo que esta informacin tiene potencial para generar una regresin lineal ya que cuenta con datos en su mayora cuantitativos y cualitativos que sospecho estn estrechamente relacionados con el rendimiento de combustible. Un anlisis pre eliminar al observar ciertas variables me di cuenta de que un modelo lineal o cuadrtico podran describir el comportamiento.

La base de datos tena algunos valores de la variable de inters como NaN, se eliminaron esas filas ya que considero que cualquier manera de conseguir ese valor aadira incertidumbre a nuestro modelo. Estaramos construyendo nuestro modelo en terreno que no es firme.

import numpy as np

# Seleccionar solo las columnas numricas
numerical_cols = df.select_dtypes(include=["float64"]).columns

# Crear un diccionario para almacenar los outliers de cada variable
outliers_dict = {}

# Calcular outliers para cada columna numrica
for col in numerical_cols:
    q1 = df[col].quantile(0.25)  # Cuartil 1
    q3 = df[col].quantile(0.75)  # Cuartil 3
    iqr = q3 - q1  # Rango intercuartil
    li = q1 - (3 * iqr)  # Lmite inferior
    ls = q3 + (3 * iqr)  # Lmite superior
    
    # Identificar los ndices de los outliers
    outliers = np.where((df[col] <= li) | (df[col] >= ls))[0]
    
    # Guardar en el diccionario si hay outliers
    if len(outliers) > 0:
        outliers_dict[col] = df.iloc[outliers]

# Imprimir resultados
for col, outliers in outliers_dict.items():
    print(f"\nOutliers en la columna: {col}")
    print(outliers)
